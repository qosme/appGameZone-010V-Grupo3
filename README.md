# ProyectoGameZone

Aplicación móvil en Kotlin para la gestión y compra de videojuegos.
Desarrollada por Eduardo Poblete y Leonardo González.

GameZone es una aplicación creada en Kotlin que permite a los usuarios explorar, comprar y administrar videojuegos, también incluye un sistema de roles con administrador y usuario, donde:
- El primer usuario registrado se crea automáticamente como administrador.
- Los usuarios posteriores se registran con rol de usuario.
- El administrador puede gestionar el catálogo de juegos y los usuarios.
- Los usuarios pueden ver el catálogo, buscar juegos y hacer compras.
